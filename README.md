# MYSKME自鸣棋

MYSKME × 英语王老师（Mr. Wang）· **课堂肉鸽自走棋** —— 单文件离线网页，扫码 / 投屏即用。

学生用答题赚来的铸币，三选一征募《狼先生与他的学生们》编年史学员组建军团；每幕演武自动演算对战，全班围观——棋子自己开打，故曰**自鸣**。答题决定经济，构筑与站位决定胜负，命运事件决定谁也别想躺赢。

**线上地址：** https://myskme.github.io/myskme-zimingqi/
**作品总目：** https://myskme.github.io/myskme-hub/

## 核心循环（一节课一局 · 30–40 分钟 · 1–8 人（含 1 对 1 课堂））

```
开局点将（人人免费领一名学员）
   ↓
┌ 答题波（老师判定：答对+2 / 难题+3 / 抢答+1 / 答错喂暴走槽）
│    ↳ 幕初公示宿敌预告：本幕演武打谁，一开始就知道
├ 征募 · 三选一（落后者先选 · 双卡合星 · 存币吃利息 · 棋位轨道按钮调站位）
├ 命运事件（20 张全局事件卡，对落后者更友好）
└ 演武 · 自动演算对战（胜者夺王座分，败者领复仇火种）
   × 三幕（谷地雾晨 → 断云之上 → 黑域门前）→ 加冕 + 战报卡 PNG
```

答错喂满「暴走槽」会触发**叶王乱入**：本幕演武取消，全员各自迎击黑域侵攻——PvP 里嵌一段全班共同的 PvE 变奏。

## 特性

- **单文件 `index.html`**，离线可用；美术资源走 `assets/` 映射层，缺图自动回退字形徽章
- 35 名编年史学员（含 S2E5 初代双璧、马将军/晰/佳，以及两批正典补全瑾/涵/浩、睿/海/欣）+ **双轴羁绊**（星象四象：厚土载物/疾风借势/烈焰同燃/潮汐同悲 × 打法两系：破云之矢/不退之誓；十二星座归四象，另有四象单环相克「土掩水→水灭火→火灼风→风蚀土」，阵营为剧情归属）+ 20 遗物 + 20 命运事件
- **MYSKME 通用游戏图标系统**：37 枚原创 24px SVG，攻击／生命／速度／铸币／王座分、站位、出售、撤销、倍速等不再依赖 Emoji 或系统图形字体；同一套资源可跨游戏复用
- 防滚雪球三件套：落后者先选卡、复仇火种、事件权重向落后者倾斜
- 黑域单位高风险高收益（航：全场最高攻，但每幕吞你 1 铸币——"自愿戴上锁"的机制化）
- 战报卡 Canvas 导出 PNG，可发家长群；localStorage 断电续局
- 老师调节阀：幕数 / 暴走阈值 / 事件开关；全程单屏投屏，题在老师手里

## 课堂提示

- 节奏慢的班：暴走阈值调 6，叶王来得更早，气氛立刻紧张
- 水平差距大：基础题多用「抢答 +1」，难题留给领先者
- 赶时间：开局选「两幕 · 快课」，战斗随时可二倍速 / 跳过

## 美术管线

`assets/` 内美术由 AI 出图工具按生产清单生成，文件名与游戏内 `ART` 映射表一一对应。新批次放入版本子目录并通过映射切换，不覆盖旧原图；缺图自动回退字形徽章。本地 file:// 打开时战报卡不嵌立绘（画布安全限制），线上部署后正常。

系统图标使用独立 UI_ICON_VER，部署副本位于 assets/ui-icons-20260714/。游戏本体为保证课堂断网与 file:// 稳定，使用同源内联 SVG；资产目录同时提供 sprite、37 个独立 SVG、manifest 与预览，方便其他作品直接复用。

## Claude / Codex 协作

两台电脑开始工作前都先读 [`AI-COLLABORATION.md`](AI-COLLABORATION.md)，从 GitHub 最新 `main` 开始；结束时记录基线 SHA、文件、资产目录和自测结果。禁止强推和用旧版 `index.html` 整体覆盖另一方的更新。

## 技术

纯 HTML/CSS/JS，零依赖、零外部请求。战斗演算核心为纯函数（`/*SIM-BEGIN*/`…`/*SIM-END*/` 标记段），可脱离 DOM 单测。
