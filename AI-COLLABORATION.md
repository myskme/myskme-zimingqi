# 自鸣棋 · Claude / Codex 协作约定

本文件是两台电脑、两个 AI 工具共同维护本仓库时的唯一交接入口。开始修改前先读本文件和 `git log -5 --oneline`；结束时追加记录，不改写或删除他人的历史条目。

## 防冲突规则

1. 每次工作都从 GitHub `main` 最新提交开始；提交前再次 `fetch`，远端前进时先 rebase/合并再推送。
2. Claude 优先负责玩法、数值、逻辑与常规接线；Codex 优先负责 Image 2 高阶美术、视觉系统、响应式视觉验收。任何一方均可修复明确的小问题，但必须在下方留痕。
3. 新美术放进独立版本目录，例如 `assets/image2-system-20260713/`；不覆盖旧原图、不删除旧资产。代码只通过 `ART` 映射和 `ART_VER` 切换。
4. 不强推，不重写历史，不执行批量清理。发现双方改到同一区段时，先保留双方意图，再做人工合并和自测。
5. 每条记录至少写：基线提交、改动文件、资产目录、自测结果、留给另一方的注意事项。
