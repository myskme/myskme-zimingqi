# MYSKME 自鸣棋 · 传世开箱大幕 20260717a

这套素材为同一开箱仪式分别设计了 iPad 横屏、iPad 竖屏和手机竖屏构图，不使用桌面图硬裁移动端。画面中央预留运行时角色卡龛位，名称、词缀、说明与点击提示全部由 DOM 叠加。

## 响应式文件

| 场景 | 1× | 2× | 3× |
| --- | --- | --- | --- |
| iPad 横屏 | `1200×840` | `2400×1680` | — |
| iPad 竖屏 | `834×1194` | `1668×2388` | — |
| 手机竖屏 | `430×932` | `860×1864` | `1290×2796` |

## 框饰与降级

- `legendary-reliquary-frame.svg`：角色展示龛位的 `currentColor` 框。
- `legendary-stage-ornament.svg`：舞台外圈的 `currentColor` 轻装饰。
- 两枚 SVG 由资源探针确认后才启用；缺失时不会出现实色遮罩。
- 任一 WebP 加载失败时，舞台退回黑金渐变，角色图失败时只移除角色图，DOM 文案、点击关闭与 4.2 秒自动关闭仍完整可用。
- `prefers-reduced-motion: reduce` 下停用入场缩放、位移与模糊动画。

浏览器运行时不读取 `manifest.json`；清单用于尺寸、文件散列和复用交接。
