## 接入约定

- 全部 SVG 均为原创几何图形，使用 `currentColor`；不含字体、脚本、外链、文字或第三方商标。
- 主要线宽为 `1.35–1.7`，无大面积实心填充；既可作 `<img>` / CSS `background-image`，也可作 `mask-image` 后由 CSS 上色。
- 图形层默认是装饰性的 `aria-hidden="true"`；按键名称、中奖姓名和操作语义必须继续由 DOM 文字与 `aria-label` 提供。
- 资产缺失时，应降级为现有 CSS 边框与 DOM 文字；不得因图形加载失败而阻断点名、计分或撤销。

## 减少动态效果

- SVG 本身不包含动画。转盘旋转、落点指针反馈和按键高亮均由 CSS 控制。
- 在 `prefers-reduced-motion: reduce` 下，转盘应直接定位结果，指针不摆动，按键不脉冲；保留静态金边、明度或尺寸反馈即可。
- 不应对 `menu-panel.svg` 设置持续旋转；它是低对比度的稳定操作台背景。
