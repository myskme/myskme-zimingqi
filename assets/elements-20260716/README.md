# 自鸣棋 · 四象元素特效 20260716a

四象每象包含 1x／2x 透明命中图集与一张高 DPR 矢量成阵环。颜色严格复用游戏现有 `ELEM_COLOR2`，不另建颜色系统。

一句话接线：基本攻击日志满足 `e.eff>1` 时，在受击单位播放攻击者所属象的 `hit`；购买后某条四象羁绊升档时，在界面过场层播放同象 `ring`。

- 命中图集：4×2、8 帧、24fps、row-major；1x 每格 128×128，2x 每格 256×256，可用 CSS `image-set`。
- 成阵环：`currentColor` SVG，建议走 CSS mask 并由 `ELEM_COLOR2[key]` 着色；需要描边动画时才安全内联。
- 128px 命中图不硬放大全屏；iPad／手机成阵始终用矢量环。
- 全盾吸收仍可显示克制元素光，同时保留护盾裂纹，二者语义不同。
- 缺图时继续保留当前“克”字、toast、飘字与 CSS 打击感。
