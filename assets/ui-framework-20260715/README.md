# 自鸣棋界面框架件 · 20260715

7 个生产 SVG 均使用 currentColor，且不含文字、字体、外链、image 或 filter。四角饰、章节分隔、云丝纹理与 MYSKME 印章属于布局装饰，**不要加入现有 45 枚 UI_ICON_PATHS**，避免破坏语义图标数量守卫。

- 角饰：64×64，四方向已分别输出。
- 章节分隔：480×32。
- 云丝纹：256×256 无缝图案；面板建议 2.5%–6% 不透明度。
- MYSKME 印章：400×400 纯路径 M 章。

在 iPad 与手机上作为 pointer-events:none、aria-hidden=true 的装饰层使用；不要缩小到 24px 语义图标。
