# 自鸣棋视觉侧收官资产 · 20260718a

本目录只新增现阶段最后一项运行美术：旧存档命中已退池事件时使用的 `ev_void`「星轨已移」事件插图。

## 运行资产

| 路径 | 尺寸 | 用途 | 接线 | 探针与降级 |
|---|---:|---|---|---|
| `events/event-void.webp` | 768×512 | 退池旧事件的展示层空转卡 | `ART.ev_void` | 不新增探针；沿用事件图 `onerror`，失败时显示「移」字形、DOM 名称与说明 |

插图不含名称、数值、说明、Logo 或水印；这些内容继续由 DOM 渲染。该视觉对象不进入 `EVENTS`，不会进入抽取池，也不会改变权重、事件流或结算。

## 长期母版

Image2 原始 PNG 为 `event-void-image2-master-1536x1024.png`，只进入长期资源包的 `masters/`，不进入 GitHub Pages。其尺寸、字节与 SHA-256 记录在 `manifest.json`。
