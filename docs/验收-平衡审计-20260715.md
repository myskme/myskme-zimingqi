## 结论

用真实代码跑了 headless 仿真（把 UNITS/RAR_STAR/BONDS/buildFighters/simulateBattle/soloBuildFoe 抽到 node 里跑，非估算）。结论：**没有"叠乘爆炸"——所有新增项都是加法（星级倍率只乘 base，词缀/星芒/淬炼/羁绊全是直加，四象1.25只乘单次伤害）。真正崩的是敌军曲线从头到尾没吃到今天任何一个新系统。**

【实测数字】
1) 敌军永远不上三星：soloBuildFoe 的升星循环 `si` 只单向走一遍 → 120 次采样、全难度全关卡，**3★敌军出现 0 次**。敌军也没有词缀/星芒/淬炼。
