## 9. 收官维护规则

1. 新资产只进新的版本化目录，不覆盖旧目录。
2. 位图先保留 Image2 PNG 母版，再导出 WebP；SVG 保持确定性、`currentColor`、无字体／脚本／外链。
3. 新 mask 必须进入对应探针，或提供不会退化成实色块的独立失败门。
4. 新响应式图源只有在全部密度档真实存在时才能进入 `ART_DPR`／`srcset`。
5. 任何新接线都要同时说明 DOM 文案、失败回退、reduced-motion 与三端裁切。
6. 不修改玩法数值、抽取权重、事件流、存档、网络、正典数据或自检玩法断言。
